# NIKO GRADNJA Website

Responsive business website for NIKO GRADNJA, a construction company focused on knauf works, ceramics, rohbau works, and final interior works.

## Overview

This project was built as a static website using HTML, CSS, and JavaScript. The goal was to create a clear, professional presentation of the company, its services, completed projects, team, contact information, and 5-year work guarantee.

## Pages

- `index.html` - homepage with animated hero section, service overview, why-us section, project preview, and contact area
- `services.html` - detailed services page for knauf works, ceramics, and rohbau
- `projects.html` - categorized project gallery with modal image preview
- `about.html` - team and company introduction

## Features

- Responsive layout for desktop and mobile screens
- Mobile hamburger navigation
- Animated preloader with company logo
- Hero image slideshow on the homepage
- Project gallery grouped by work category
- Modal image viewer for project photos
- Contact section with phone, email, location, and WhatsApp link
- Basic SEO metadata for main pages

## Technologies

- HTML5
- CSS3
- JavaScript

## Project Purpose

The website was created as a freelance-style business project and portfolio piece. It demonstrates page structure, responsive design, image-based layout, navigation behavior, gallery interaction, and practical business content organization.
